# 🚗 Smart EV Charging Station

A Java console-based application that simulates a Smart EV Charging Station. The project allows users to enter vehicle details, select a charging station, choose a charging type, calculate charging cost, and generate a receipt.

## ✨ Features

- Vehicle Information Input
- EV Charging Station Selection
- Fast Charging / Normal Charging
- Charging Cost Calculation
- Receipt Generation
- Console-Based User Interface

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- VS Code

## 📂 Project Structure

SmartEVChargingStation/
│── Main.java
│── Vehicle.java
│── ChargingStation.java
│── ChargingCalculator.java
│── Receipt.java

## ▶️ How to Run

1. Open the project in VS Code.
2. Compile the Java files.
3. Run `Main.java`.
4. Enter the required vehicle and charging details.
